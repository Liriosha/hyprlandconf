# Install
`cd ~/.config && rm -rf hypr && git clone https://github.com/Liriosha/hyprlandconf hypr`
>[!WARNING]
> If you have anything important in ~/.config/hypr back it up before running this command!

Restart hyprland using META+M and log in again


# Dependencies

```
ttf-jetbrains-mono-nerd fuzzel hyprlock hypridle awww mako waybar hyprpolkitagent xdg-desktop-portal-hyprland pamixer waypaper matugen wl-clipboard pulseaudio pipewire-pulse hyprland hyprlock hypridle rose-pine-hyprcursor
```
