# Install
`cd ~/.config && rm -rf hypr && git clone https://github.com/Liriosha/hyprlandconf hypr`
Restart hyprland using META+M and log in again
# Dependencies

```

fuzzel hyprlock hypridle hyprpaper mako waybar hyprpolkitagent xdg-desktop-portal-hyprland

```
Please install through your package manager
